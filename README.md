This is a simple JS implementation of web-based text RPG game.

# Usage
1) Clone repo:
```
git clone https://github.com/evelasco93/javascript-rpg.git
```
2) Run the site as local host or use live server extension to test the game locally.

3) Enjoy repelling your first dragon!

Feel free to use the code as needed, this was based off of this project: https://replit.com/@BeauCarnes/JavaScript-RPG-Starter#index.html
